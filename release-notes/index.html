v2.1 Build 55 [+][Tables] Добавлена возможность редактирования формулировки вопросов и названия альтернатив в специальном окне, которое вызывается пунктом «Редактировать» в выпадающем меню по правой клавише мыши на списке вопросов. В окне можно увидеть поле для редактирования текста вопроса и список альтернатив. Изменения вносятся в массив по нажатию кнопки «Сохранить» или по кнопкам со стрелками (в таком случае будет осуществлен переход на предыдущий или следующий вопрос). Так же можно через буфер обмена внести изменения в весь список альтернатив. Чтобы отменить неправильные изменения, нужно закрыть окно редактирования или правой клавишей по списку альтернатив и выбрать меню «Обновить». После редактирования массив обязательно необходимо сохранить (Главное меню – FRM – Сохранить FRM массив).

v2.1 Build 53 [+][Chart] Построение чартов теперь учитывает взвешивание и фильтр (без сплитов) [+][Chart] Добавлена возможность построения средних и топбоксов в разбивке. Все вопросы идут один за другим горизонтально. При использовании данной опции, все вопросы должны иметь один и тот же список альтернатив. Для средних можно использовать настройку «считать средние только для выбранных альтернатив» из основных таблиц. [+][Chart] Добавлена кнопка для вывода всех признаков в один чарт, вертикальное расположение. Например, можно вывести вопросы «Спонтанное знание марки 1-упоминание», «Спонтанное знание марки», «Знание марки с подсказкой» в один чарт, т.е. это фактически одномерки, вертикально, одна за другой. При использовании данной кнопки, все вопросы должны иметь один и тот же список альтернатив. [][Tables] Мелкие недоточеты. [][Tables] При прописывании альтернатив в топбоксах можно использовать запись с минусом, типа 1-10, что будет означать все альтернативы подряд с первой по десятую. [+][Tables] При сравнении значимости различий средних с тоталом, ячейки подкрашиваются двумя разными цветами – для среднего больше-равного тотальному среднему и меньше. Цвет подкрашивания регулируется в файле «table.css» в папке программы CSS. [+][Output] Добавлен выбор способа раскраски таблиц. Сделал парочку цветовых схем.
